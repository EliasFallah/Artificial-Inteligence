Assignment 1 - Artificial Intelligence

This repository contains the code and resources for Assignment 1.

How to Run the Code
-------------------

1. Install Dependencies

Before running any code, ensure you have all required Python packages installed. Run the following command in your terminal:

    pip3 install -r requirements.txt

2. Run the Assignment Code

You can run each part of the assignment using the following commands:

- To run the 8-Tile Puzzle solver:
  
      python3 8Tile.py

- To run the Sliding Tile Puzzle solver:
  
      python3 SlidingPuzzle.py

Notes
-----

- Make sure you are using Python 3.
- All dependencies are listed in `requirements.txt`.
- If you encounter any issues, please check that all dependencies are installed and that you are in the correct directory.
